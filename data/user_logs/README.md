# User Logs

本目录用于存放用户真实交易日志（长期累积数据）。

## 命名建议
- `UserSummary_YYMMDD.md`，例如：`UserSummary_260305.md`

## 写作频率
- 每个交易日 1 篇（推荐收盘后写）
- 若出现重大交易决策变化，可追加补充日志

## 内容建议
- 今日状态（emotion/stress/focus）
- 市场背景（regime/key_var）
- 交易记录（BUY/SELL 明细和理由）
- 扫描与风险（anxiety/fomo/not_trade）
- 复盘反思（fact/gap/lesson）
- 可选 `@AI` 指令（告诉系统你希望重点检查什么）
