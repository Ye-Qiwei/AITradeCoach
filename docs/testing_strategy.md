# 测试策略

## 单元测试
- 日志解析与字段级错误
- 交易流水推演和 PnL 推导
- 记忆召回过滤与排序
- 窗口选择逻辑可解释性
- 输出 schema 校验

## 集成测试
- 每日链路端到端（含占位 MCP）
- 多日志样本覆盖（短线/中期/长期 thesis）
- 不同市场情景（事件冲击/震荡/趋势）

## 回放测试
使用历史日志 + 事后市场结果，验证：
- 偏差识别能力
- 超前洞察识别能力
- 不因短期不涨误判长期 thesis
- follow-up 推荐是否合理

## 测试资产
- 单元测试：schema、日志解析、记忆层、窗口选择、PromptOps、Prompt registry
- 集成测试：每日主链路、记忆写回、MCP 模拟返回
- 回放测试：ReplayEvaluator/ReplayRunner 与 case 评分机制
